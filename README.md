# OnlineHospitalManagementSystem
PHP Based Hospital Management System

# Procedure to setup

- Extract file
- Copy the main project folder
- Paste in xampp/htdocs/

- Open a browser and go to URL “http://localhost:8080/phpmyadmin/”
- Then, click on the databases tab
- Create a database naming “onlinehospitalmanagementsystem” and then click on the import tab
- Click on browse file and select “onlinehospitalmanagementsystem.sql” file which is inside the “backup” folder
- Click on go.

# After Creating Database,

- Open a browser and go to URL “http://localhost:8080/OnlineHospitalManagementSystem/”
